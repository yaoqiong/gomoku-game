# 五子棋游戏

一个基于 HTML5 Canvas 的网页版五子棋（Gomoku）游戏，无需任何依赖，打开浏览器即可畅玩。

## 项目简介

双人对战的五子棋游戏，黑方先手，双方轮流落子，率先在棋盘上连成五子（横、竖、斜方向均可）的一方获胜。

## 技术栈

- **HTML5 Canvas** — 棋盘与棋子的绘制渲染
- **原生 JavaScript** — 游戏逻辑（零框架依赖）
- **CSS3** — 页面样式与布局

## 项目结构

```
gomoku-game/
├── index.html   # 页面入口
├── game.js      # 游戏核心逻辑
└── style.css    # 样式定义
```

## 游戏规则

- 棋盘大小：15 × 15
- 黑方先手，双方交替落子
- 横、竖、斜任意方向连续 5 颗棋子即获胜

## 快速开始

无需安装任何依赖，直接用浏览器打开 `index.html` 即可：

```bash
# 直接双击文件，或通过命令行打开
open index.html
```

## 功能特性

- 黑白双方轮流落子，实时切换当前玩家提示
- 自动检测胜负，胜利后显示结果提示
- 支持一键重新开始
